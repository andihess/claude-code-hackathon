# KB-2001 — VPN slow or dropping connection

**Category:** network

## When this applies
User can connect to the VPN but performance is degraded — slow throughput,
high latency, or intermittent disconnects. A workaround usually exists, so this
is rarely a top-severity issue for a single user.

## Workaround / resolution
1. Switch to the nearest VPN gateway region in the client.
2. Reconnect over a wired connection or a different network if possible.
3. Clear the client's saved profile and re-download it.
4. If many users on the same site are affected, treat as a broader network issue.

## Severity guidance
**P3** for a single user with a workaround (degraded but working). Raise toward
**P1/P2** only if a whole team or site is blocked with no workaround — that is scope,
not just the word "urgent." Route unresolved cases to `network-ops`.
